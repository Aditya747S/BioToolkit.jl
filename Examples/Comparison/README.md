# Comparison Benchmarks

This directory contains small, runnable Julia benchmarks that compare BioToolkit against other Julia packages and, where relevant, Python baselines.

## Current scripts

- `fasta_index_compare.jl`: FASTA indexing and random access against a Python baseline
- `fastq_compare.jl`: FASTQ parse and write throughput
- `hamming_compare.jl`: Hamming distance against a Python baseline
- `sequence_toolkit_compare.jl`: sequence helpers against Biopython
- `motif_pwm_compare.jl`: motif counting and PWM scoring against Biopython
- `pairwise_align_compare.jl`: pairwise alignment against Biopython
- `pairwise_affine_compare.jl`: affine-gap alignment against Biopython
- `protein_matrix_compare.jl`: named substitution matrices and protein alignment against Biopython
- `flatfile_compare.jl`: EMBL flat-file parsing against Biopython
- `abif_compare.jl`: ABIF trace parsing benchmark
- `expanded_biopython_compare.jl`: expanded Biopython overlap benchmark covering melting temperature, slicing, BLAST parsing, structure, phylogenetics, and big-data parsers
- `msa_compare.jl`: multiple sequence alignment round trips against Biopython
- `msf_compare.jl`: MSF round trips against Biopython
- `motif_scan_compare.jl`: both-strands motif scanning against Biopython
- `midpoint_root_compare.jl`: midpoint rooting against Biopython
- `consensus_support_newick_compare.jl`: consensus construction and support-labeled Newick output against Biopython
- `prune_reroot_compare.jl`: pruning and outgroup rerooting against Biopython
- `graph_export_compare.jl`: DOT/Mermaid export helpers against closest Biopython tree rendering/serialization baselines
- `mmcif_compare.jl`: mmCIF parsing against PDBTools on a downloaded RCSB structure
- `bed_compare.jl`: full-file BED parsing against BED.jl on a downloaded UCSC interval track
- `window_coverage_compare.jl`: windowed coverage against a Python baseline
- `cuda_histogram_compare.jl`: Julia CPU, Julia CUDA, and Python histogram comparison
- `expanded_biopython_compare.jl`: expanded Biopython overlap benchmarks for sequence utilities, BLAST parsing, structure, phylogenetics, and VCF/K-mer baselines
- `expanded_julia_only_compare.jl`: Julia-only structure, phylogenetics, VCF, K-mer, and mocked Entrez batch throughput checks
- `julia_package_compare.jl`: Julia-vs-Julia comparisons against BioSequences, BioAlignments, BED, and GFF3
- `longread_recovery_benchmark.jl`: synthetic long-read SV truth generation and recovery table output
- `spatial_recovery_benchmark.jl`: synthetic spatial mixture deconvolution recovery table output
- `pipeline_recovery_benchmark.jl`: deterministic DAG recovery and cache consistency table output
- `coevolution_recovery_benchmark.jl`: synthetic coevolution/contact recovery table output
